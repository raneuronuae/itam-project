# IT Asset Management (ITAM) System

A robust web-based solution designed for efficient IT asset tracking, inventory management, and asset lifecycle oversight.

## 🔑 Key Features
* **Asset Tracking:** Monitor location and condition in real-time.
* **Procurement:** Streamlined PO and Goods Receipt workflows.
* **Maintenance:** Automated repair request and history logging.
* **Role-Based Access:** Secure management for Admins, Managers, and Viewers.

## 🚀 Quick Setup
1. **Clone the repo:** `git clone https://github.com/raneuronuae/itam-project.git`
2. **Install requirements:** `pip install -r requirements.txt`
3. **Run migrations:** `python manage.py migrate`
4. **Start server:** `python manage.py runserver`

## 🛠 Tech Stack
* **Backend:** Python / Django
* **Admin Interface:** Django Jazzmin
